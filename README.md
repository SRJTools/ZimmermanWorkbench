# Zimmerman Workbench

Zimmerman Workbench is a Windows Digital Forensics and Incident Response (DFIR) platform that streamlines forensic investigations by integrating and correlating artifacts collected from the Eric Zimmerman tool suite.

The platform helps investigators transform raw forensic artifacts into actionable findings through artifact correlation, timeline analysis, Sigma-based detections, advanced searching, and investigation workflows.

---

## Features

### Artifact Processing

Supports forensic artifacts generated by Eric Zimmerman tools, including:

* Windows Event Logs (EVTX)
* Master File Table (MFT)
* Registry Hives
* Prefetch
* Amcache
* ShimCache
* Jump Lists
* LNK Files
* ShellBags

---

### Automatic Tool Discovery

Zimmerman Workbench automatically discovers supported Eric Zimmerman executables, including those located in nested subfolders.

Supported tools include:

* MFTECmd
* EvtxECmd
* RECmd
* PECmd
* JLECmd
* LECmd
* SBECmd
* AmcacheParser
* AppCompatCacheParser

---

### Correlation Engine

The platform correlates evidence across multiple artifacts to help identify suspicious activity.

Examples:

* Process execution validation
* File activity correlation
* Registry persistence correlation
* Multi-artifact evidence linking

---

### Sigma Detection Support

Built-in Sigma rule support allows investigators to identify suspicious events and known attack techniques.

Examples:

* Service Creation
* PowerShell Abuse
* Scheduled Tasks
* Persistence Mechanisms
* Privilege Escalation Indicators

---

### Advanced Search Engine

Search across all imported artifacts from a single interface.

Supported queries:

```text
eventid:4624

process:powershell.exe

user:administrator

artifact:evtx

"failed login"

eventid:4624 AND user:administrator

powershell AND NOT defender
```

---

### Unified Timeline

Investigators can view events from multiple evidence sources in a single timeline to reconstruct attack activity and system events.

---

### Findings Engine

Automatically generates investigation findings based on:

* Correlation results
* Sigma matches
* Suspicious artifact activity

---

### Investigation Graph

Visualize relationships between:

* Users
* Processes
* Files
* Registry Entries
* Events

to better understand attack chains and system activity.

---

### Reporting

Generate investigation reports containing:

* Findings
* Timeline Data
* Artifact Summaries
* Sigma Matches
* Correlation Results

---

## Requirements

* Windows 10 / Windows 11
* Eric Zimmerman Tools
* Administrator privileges recommended
* Microsoft Visual C++ Redistributable (if required)

---

## Installation

1. Download the latest release.
2. Run the installer:

```text
Zimmerman DFIR Workbench-2.6.0-win64.exe
```

3. Launch Zimmerman Workbench.
4. Configure or scan for Eric Zimmerman tools.
5. Create a case and begin analysis.

---

## Typical Workflow

1. Create Case
2. Import Evidence
3. Run Artifact Collection
4. Parse Artifacts
5. Execute Sigma Detection
6. Review Findings
7. Investigate Timeline
8. Export Report
